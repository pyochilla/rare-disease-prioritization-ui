# rare-disease-prioritization-ui

An interpretable, user-friendly workflow for rare-disease prioritization.

## Overview
This repository contains a browser-based prototype for rare-disease prioritization from partial gene and phenotype cues.

## Features
- Gene set input
- Phenotype cue input
- Ranked disease output
- Diagnostic neighborhood interpretation
- Conditional expansion guidance

## Disclaimer
This prototype is intended for workflow demonstration and research communication. It is not a validated clinical diagnostic system.
